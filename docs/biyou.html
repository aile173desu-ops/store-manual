P1 基本情報

03

カウンセリング

P5 を使用 ― 質問形式で深掘り

P5 ｜ カウンセリング質問 (Q1〜Q8)
Q1. ご来店のきっかけ
何がきっかけでご来店いただきましたか？（紹介・SNS・検索など）
Q2. 一番気になる箇所
お顔で一番気になっている箇所はどこですか？
・エラの張り ・頬骨 ・顎のライン ・左右差
Q3. いつ頃から気になっていますか？
お悩みが始まった時期と、変化の経過を確認します。
Q4. 過去の美容施術経験
小顔矯正やフェイシャル等の施術を受けたことはありますか？
・効果の実感 ・施術内容 ・満足度
Q5. 日常のクセ・習慣
噛み癖（左右どちら）、頬杖、うつ伏せ寝などの習慣はありますか？
・食いしばり ・歯ぎしり ・姿勢の癖
Q6. 理想のゴール
施術を通じてどうなりたいですか？具体的なイメージを共有します。
Q7. 通院可能な頻度
5〜10日に1回のペースをお勧めしつつ、ご都合を確認します。
Q8. ご予算感
月々のご予算感をお聞きし、最適なプランをご提案します。
約8分を目安に進めましょう。お客様の話を傾聴しつつ、必要な情報を収集します。

P6

骨格図（骨と悩みの対応表）

骨格の部位歪むと出る悩み
側頭骨（こめかみ）エラ張り・食いしばり・頭痛・こめかみの張り
頬骨（頬の土台）ほうれい線・頬のたるみ・顔の左右差
前頭骨（おでこ）眉毛・目の高さが左右でズレる・目が重たく見える
下顎骨（あご）フェイスラインの崩れ・あごのズレ・噛み合わせのバランス
スクリプト — PART A（骨格と悩みの因果を伝える）
「こちらの図を見てください。お顔はいくつかの骨で構成されています。例えば、このこめかみの骨（側頭骨）がずれると、エラが張ったりこめかみが張ったりするんですよ。頬骨がずれると、ほうれい線や頬のたるみが出てきます。おでこの骨（前頭骨）がずれると、眉毛や目の高さが左右でズレてきます。つまり、お顔のお悩みは骨格の歪みから出てきているものがほとんどなんです。」

PART B｜先ほど撮った写真で実感させる

  1. 「では、先ほど撮った今の写真を見ていきましょう。」
  2. 「パッと見て、気になるところはありますか？」
    何かしら答えてもらえるのを待つ。急がず反応を引き出す
  3. 「実は、ここの部分もよく見てみてください。眉毛の高さ、目の開き方、ほうれい線の出方…全体的に少し右（左）が下がっているんですよ。」
  4. 「○○さんの場合は、ここ（例：側頭骨・頬骨）の骨が歪んでいるので、やっぱりこういう悩みが出てきていますよね。さっきツールで見ていただいた通りで、骨格の歪みがそのままお悩みにつながっているんです。なんとなくイメージできましたか？」
患者さんに「気になるところ」を先に言ってもらうことで、説明が「押しつけ」でなく「共感・確認」の形になる。これが納得感の核心。

施術（首 → 頭蓋骨 → 骨格矯正の順）
ツールなし ｜ 9鍵：③情熱・④メンタル
  1. 施術前説明
    「筋肉を柔らかくするために一番大切なのは血を流すことです。顔への血流は必ず首を通るので、まず首から緩めます。その後、頭の骨 → 顔の骨格の順で整えていきます。部位によって痛みが出ることがあります。つらい時はすぐ言ってください。」
  2. 首ほぐし後 体感確認
    「もう一回同じところを触ってみてもらっていいですか？さっきと比べて柔らかくなっているのわかりますか？まだ顔には直接触れていないのに、顔まわりも少し柔らかくなっているはずです。それだけ首の硬さが顔に影響しているんですよ。」
  3. 頭の矯正
    「では骨格を直接動かしていきます。まず頭から始めます。（施術中）動いている感じはわかりますか？これが骨格が直接動いている感覚です。硬い部位は痛みが出やすいです。無理しなくていいので遠慮なく言ってください。」
  4. 頭の矯正後 ビフォーアフター確認
    「頭の矯正が終わりました。さっきと比べて、頭のサイズ感や頬の高さが変わっているのわかりますか？」

    「次は頬骨の矯正をしていきます。頬骨を整えることで、ほうれい線・たるみ・顔の左右差が改善していきます。」
  6. 頬骨矯正後 ビフォーアフター確認
    「頬骨の矯正が終わりました。右と左で頬の高さや張り感、変化しているのわかりますか？」
ポイント：細かくビフォーアフター確認を入れる
施術の各工程ごとに「さっきと比べてどうですか？」と確認を挟む。患者さん自身に変化を実感してもらうことで納得感と信頼感が高まる。
骨格矯正は最初は痛みが出ることがある。「通うほど痛くなくなります」と伝えて継続動機につなげること。

施術中の圧痛確認（マッサージ中の痛みの説明）

「今、僕がマックス10の強さで押せるとしたら、現在押している強さは2ぐらいです。なぜかと言うと、筋肉は硬ければ硬いほど押された時に痛みを感じますが、逆に柔らかくなれば痛みはほとんど感じなくなります。そのため、理想は『7割ぐらいの強さで押しても痛くない状態』です。」
圧痛の説明をすることで、患者さんが「痛い＝悪い状態」と理解でき、改善のモチベーションにつながる。また、施術が進むにつれて痛みが減ることを先に伝えておくことで継続動機になる。
